# Kutuphane Yonetim Sistemi Projesi 1. Hafta Raporu

VERDİĞİM PROJE ÖN RAPORUNDA BELİRTTİĞİM 1. HAFTA MADDELERİNİ TAMAMLADIM.

BU HAFTA ŞU ŞEKİLDE İLERLEDİM:
- POSTGRESQL TARAFINDA `USERS`, `BOOKS`, `MEMBERS`, `LOANS`, `LOANREQUESTS` TABLOLARINI OLUŞTURDUM.
- NODE.JS + EXPRESS İLE BACKEND API ALTYAPISINI KURDUM.
- VERİTABANI BAĞLANTISINI TEST ETTİM VE BACKEND İLE POSTGRESQL ARASINDA İLETİŞİM KURDUM.
- İLK CRUD ENDPOİNTLERİNİ TAMAMLADIM:
  - `GET /APİ/BOOKS`
  - `POST /APİ/BOOKS`
- `BOOKS` TABLOSUNA 10 ÖRNEK KİTAP EKLEDİM.
- REACT NATİVE TARAFINDA ANA EKRAN VE BASİT KİTAP LİSTESİ EKRANINI OLUŞTURDUM.
- REACT NAVİGATİON KURULUMUNU YAPIP EKRANLAR ARASI GECİŞİ AKTİF ETTİM.

BU HAFTA YAZDIĞIM KODLARIN NE YAPTIĞI:
- BACKEND KODU, KİTAP VERİLERİNİ POSTGRESQL VERİTABANINDAN ÇEKİP JSON OLARAK DÖNÜYOR.
- YENİ KİTAP EKLEME ENDPOİNTİ İLE GELEN VERİYİ DOĞRULAYIP VERİTABANINA KAYDEDİYOR.
- MOBİL UYGULAMA BACKEND'DEN KİTAP LİSTESİNİ ÇEKİP EKRANDA KART YAPISINDA GÖSTERİYOR.
- ANA EKRANDAN KİTAP LİSTESİNE GEÇİŞ, REACT NAVİGATİON STACK ÜZERİNDEN YAPILIYOR.
